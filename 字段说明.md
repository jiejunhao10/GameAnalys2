# duckdb数据说明
用到的表是在 ga2_dwd


## table字段说明
### ga2_dwd.dim_role
    rid	varchar	角色id
    account	varchar	游戏账号
    openid	varchar	渠道账号
    svrid	varchar	区号
    pid	varchar	渠道标识
    create_account_date	varchar	创建账号日期
    create_role_date	varchar	创建角色日期
    first_login_date	varchar	首次登录日期
    first_recharge_date	varchar	首次充值日期
    first_recharge_money	int	首次充值金额
    is_old_player	int	是否老玩家

### ga2_dwd.fact_player_login_recharge
    rid	varchar	角色id
    name	varchar	角色名
    min_lv	int	最小等级
    max_lv	int	最大等级
    stat_date	varchar	时间_天
    continue_login_date_cnt	int	连续登陆天数
    login_cnt	int	登录次数
    logout_cnt	int	登出次数
    online_time_sec	int	在线时长(秒)
    real_cnt	int	实际充值次数
    real_money	int	实际充值金额
